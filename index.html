HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Semantic elements like "section" divide the visible content into chunks that can be referenced as a whole unit.
  • H1 heads should be used only one time on each page.

CSS

  • The rel attribute describes the relationship between the HTML file and the linked file.
  • "box-shadow" is a predefined property of an HTML element.
  • Margins are the space outside the element's bounding box; padding is the space inside the element's box.
  • A period precedes a class selector.
  • The MDN Web Docs on CSS website describes additional properties and values.

Git

  • git status checks which branch we are currently on.
  • git checkout -b [branch-name] creates a new branch and switches to it.
  • git switch [branch-name] switches to [branch-name].
  • git commit -m "[description of changes]" commits changed files to the feature branch.
  • On a Mac, the ".ds_store" file might be problematic.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • I have a lot of experience with functions already from my work with VBA.
  • My biggest challenges are going to be:
    • Knowing when to use braces, brackets, and parens.
    • Learning the vocabulary of JavaScript generally and of the particular “flavor” of JavaScript that I will use in a future job.
